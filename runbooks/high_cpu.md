# High CPU Troubleshooting Runbook
## Objective
Investigate and mitigate high CPU usage.
## Steps
1. Check CPU usage: `echo CPU usage: 23%`
2. Identify top CPU-consuming processes: `echo top process: python3 (12%), nginx (8%)`
3. Check system load: `echo system load: 0.45 0.38 0.32`
4. Restart affected service: `echo service restarted successfully`
5. Verify CPU usage has decreased: `echo CPU usage: 11% - within normal range`
